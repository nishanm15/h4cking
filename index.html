  • Beginner

    Host discovery (Ping sweeps, ARP scans), Port Scanning (TCP Connect, SYN Stealth, UDP, Xmas, FIN, Null), Service & Version Detection, OS Detection fundamentals.

    Nmap Scan Techniques
    THM: Nmap Live Host Discovery
  • Intermediate

    Master common flags: `-p-` (all ports), `-sV` (version), `-O` (OS), `-sC` (default scripts), `--script` (specific scripts/categories like `vuln`), `-T<0-5>` (timing), `-oN`/`-oX`/`-oG` (output), `-Pn` (no ping), `-n` (no DNS).

    Nmap Manual
    Nmap Scripting Engine (NSE) Docs
    THM: Further Nmap
  • Beginner

    Understand automated scanning to identify known vulnerabilities (CVEs) based on detected services, versions, and configurations. Differentiate from port scanning. Awareness of false positives/negatives.

    OWASP Vuln Scanning Tools
    Conceptual / Covered in tool usage
  • Intermediate

    Install and configure Nessus Essentials (free) or OpenVAS/GVM. Run basic scans, interpret reports, understand severity ratings, and identify potential high-impact findings.

    Nessus Essentials
    Greenbone (OpenVAS)
    THM: Nessus Redux
  • Intermediate

    Identify open SMB ports (139, 445). List shares (`smbclient -L`, `nmap smb-enum-shares`), check for anonymous/guest access, list files, potentially download/upload. Tools: `enum4linux-ng`, `smbmap`.

    HackTricks SMB Pentesting
    enum4linux-ng
    THM: Attacking Neptune (SMB Example)
  • Intermediate

    Identify open NFS port (2049). List exported shares (`showmount -e target_ip`). Check share permissions (read/write, no_root_squash). Mount shares locally (`mount -t nfs`). Tools: `nmap nfs-* scripts`.

    HackTricks NFS Pentesting
    THM: Basic Pentesting (NFS Example)
  • Intermediate

    Identify open SMTP port (25, 465, 587). Use commands like `VRFY` (verify user) and `EXPN` (expand mailing list) if enabled. Tools: `nmap smtp-commands`, `smtp-user-enum`.

    HackTricks SMTP Pentesting
    SMTP Cheatsheet
  • Intermediate

    Identify open SNMP port (161 UDP). Brute-force community strings (e.g., 'public', 'private'). Walk the MIB tree to gather system info, network config, user accounts. Tools: `snmpwalk`, `snmp-check`, `onesixtyone`.

    HackTricks SNMP Pentesting
    THM: Agent Sudo (SNMP Example)
  • Intermediate

    Identify web server software/version (Nmap, headers). Find hidden directories and files using brute-forcing tools (`gobuster`, `feroxbuster`, `dirsearch`, `ffuf`, `dirb`) with wordlists (e.g., `dirbuster`, `SecLists`). Fingerprint web technologies.

    SecLists (Wordlists)
    gobuster / feroxbuster
    THM: GoBuster
  • Intermediate

    Use Nikto to perform comprehensive web server scans, identifying outdated software versions, dangerous files/CGIs, and other potential security issues based on its database.

    Nikto (GitHub)
    THM: Web Scanning (covers Nikto)
